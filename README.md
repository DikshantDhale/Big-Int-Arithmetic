# Big Integer Arithmetic Library (C++)

A C++ implementation of arbitrary precision integer arithmetic using a custom BigInt class.

## Features
- Supports integers larger than built-in C++ types
- Operator overloading for arithmetic operations
- Modular class design
- Separate header and implementation files
- Makefile based build system

## Files

BigInt.h – class definition  
BigInt.cpp – implementation of big integer operations  
main.cpp – sample program demonstrating usage  
Makefile – build configuration

## Compile

make

## Run

./BigInt
